# savethem

Elektroniczny rozum, wyposażony w skromny arsenał pytań, przemierza nieznane terytorium w poszukiwaniu narzędzi, których istnienia się domyśla, lecz których natury jeszcze nie zna. Kiedy już je znajdzie — a znajdzie, bo skonstruowano go właśnie w tym celu — powierzy obliczenia maszynie, sam zaś zachowa godne milczenie rozumu rozumiejącego swoje granice.

Cel: dotrzeć do Skolwina. Zasoby: skończone. Czas: również.

```bash
npm install
cp .env.example .env  # uzupełnij klucze
npm run build
npm start
```
